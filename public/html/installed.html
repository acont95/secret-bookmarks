Secret Bookmarks

Welcome to Secret Bookmarks! This extension provides the ability to password encrypt the bookmark entries inside of a bookmark folder.

Getting Started

  1. Make a Backup

    Using your browsers Bookmark Manager, export your bookmarks to ensure your data is protected before proceeding.

  2. Access the Extension

    Secret Bookmarks is accessed by default from the navigation bar alongside the URL Bar using the icon.

  3. Encrypt a Folder

    Your bookmark folder hierarchy is listed in the popup panel. To encrypt a folder toggle the slider next to the folder name and set a password.

  4. Decrypt a Folder

    To decrypt a bookmarks folder, select Secret Bookmarks in the naviagtion bar and toggle the slider next to the the encrypted folder. You will be prompted for your password.

  5. Backup Encryption Details

    The recommended method of backing up your bookmarks is to periodically export your unencrypted bookmarks using the browsers bookmarks manager. In addition, Secret Bookmarks provides the ability to export encryption details of your managed folders to a text file. This file can be imported to restore settings, but note that current bookmark folders settings will be overwritten by the imported data if the same folder is referenced.

  6. Add or Move a Bookmarks

    Adding or moving an existing bookmark to a managed folder will automatically encrypt the bookmark if the folder is currently locked. Removing an encrypted bookmark from a managed folder will NOT prompt for the password and decrypt the bookmark.

  7. Change a Folders Passphrase

    Select the icon, enter your current password and a new password then select submit.

  8. Delete a Folders Encryption Keys

    Select the icon, enter your current password and select Delete Keys.

Notes

  • Applicable Bookmark Entries

    Secret Bookmarks uses Data URLs to store the base64 encrypted bookmark URL. The extension uses the presence/absence of a data URL to determine if a bookmark is already encrypted and when to decrypt a bookmark. As a result of this implementation detail, user supplied data URLs bookmarks are not supported with this extension and may cause undefined behavior. In addition, moving an encrypted bookmark from one folder to another encrypted folder will cause decryption failures.

  • Extension Settings

    Secret Bookmarks will use sync storage if available to synchronize encryption details, otherwise storage will fallback to local storage. When using sync storage, keep in mind adding a bookmark to an encrypted folder on a device without Secret Bookmarks installed may result in unencrypted bookmarks existing alonside encyrpted bookmarks in a folder believed to be in a locked state. You may need to decrypt then encrypt the folder again to ensure all bookmark entries are encrypted.

    Information on the Secret Bookmarks technical implementation can be found on GitHub.
